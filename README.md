## 1️⃣ Automated Greenhouse System

### Description
An automated greenhouse system that monitors and controls temperature, soil moisture, and light intensity to maintain optimal plant growth conditions.

### Features
- Automatic temperature-based fan control
- Soil moisture–based irrigation
- Light intensity–based lighting control
- Fully autonomous operation

### Tech Stack
- Arduino Uno
- DHT11 / DHT22
- Soil Moisture Sensor
- LDR
- Relay Module
- Embedded C++

### File
- `automated_greenhouse.cpp`
